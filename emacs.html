Kısaca GNU Emacs

GNU Emacs bir metin editörü ama metin editörün den beklenmeyen her şeyi yapabiliyorsunuz mesela oyun oynayabiliyorsunuz maillerinizi okuyabiliyorsunuz.Sanırım bu kadar çok konuşulmasının sebebide bunlar.

GNU Emacs Programcılar için birebir bununda sebebini kullanmaya başlarsanız anlayacaksınız elinizi klavyeden hiç çekmeden her şeyi yapabiliyorsunuz mouse'a ihtiyacınız kalmıyor.

Ubuntu'da sudo apt-get install emacs

Arch Linux'da sudo pacman -S emacs

Fedora'da sudo yum install emacs

Open Suse'de sudo zypper install emacs

kodlarıyla kurulumu yapıyoruz. Sonra GNU Emacs'i çalıştıralım. CTRL + X Yaptıktan Sonra CTRL + F yapıp yeni dosya oluşturmak istiyorsak dosya ismini bir text dosyasını açmak istiyorsak o text dosyasının ismini yazmamız yeterli.

Ayrıca bu verdiğim bağlantıda aşağıda verdiğim kısayolların örneklerle uygulaması var.
